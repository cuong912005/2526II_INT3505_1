# CLIENT-SERVER - Tách biệt client và server

## Kịch bản
Client (HTML/JS) và Server (Flask API) hoạt động độc lập. Client gọi API từ server qua HTTP.

## Cài đặt
```bash
pip install flask-cors
```

## Chạy
```bash
# Terminal 1: Chạy server
python server.py

# Terminal 2: Mở client.html bằng browser
```

## Test
Mở client.html sẽ tự động fetch data từ server API.
